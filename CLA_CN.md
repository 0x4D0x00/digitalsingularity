# 贡献者许可协议 (CLA)

感谢您对 Silicoid Core — PotAGI 后端的兴趣。为了确保项目能够继续在其当前条款下维护，所有贡献者必须同意本贡献者许可协议。

## 协议概述

通过向 Silicoid Core 提交贡献，您同意以下条款：

### 1. 权利授予

您特此授予项目所有者（0x4D）永久、全球性、非独占、免版税的许可，以根据以下条款使用、修改、分发和再许可您的贡献：

- 项目当前许可条款（Business Source License）
- 项目所有者可能采用的任何未来许可条款
- 商业许可安排

### 2. 知识产权

您声明并保证：

- 您是贡献的唯一作者
- 您有权授予上述许可
- 您的贡献不侵犯任何第三方知识产权
- 您已获得包含在您的贡献中的任何第三方内容的必要许可

### 3. 署名

虽然项目所有者保留商业许可权利，但您将被认可为贡献者：

- Git 提交历史
- 项目文档
- 发布说明（由项目所有者决定）

### 4. 无额外限制

您同意不在项目或其用户已建立的许可条款之外，对项目或其用户施加任何额外的许可限制或要求。

### 5. 适用法律

本协议应受中国法律管辖，并以杭州作为争议管辖地。

## 如何签署本协议

要为 Silicoid Core 做贡献，您必须通过以下方式表明您同意这些条款：

1. **代码贡献**：在您的拉取请求描述中包含以下声明：

   ```
   我已阅读并同意 Silicoid Core 的贡献者许可协议 (CLA.md)。
   ```

2. **其他贡献**：发送电子邮件至 moycox@Outlook.com，主题为 "Silicoid Core CLA 协议"，并包含：
   - 您的全名
   - 您的 GitHub 用户名（如适用）
   - 您已阅读并同意本 CLA 的声明
   - 您的贡献描述
